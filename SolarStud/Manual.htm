1. Introduction

SolarStudio   is a modern and robust program for a tanning studio. It's main features are ease of use, stylish design, easy installation and maintenance. The program can control up to 6 solariums (optionally up to 16 or up to 32). In addition to controlling the solariums, there is a tiny database for the cosmetics and other accessories which can be sold in a tanning studio with a cash desk function. With SolarStudio  you can use 3 ways for payment for a tanning session: cash payment, member's card or chip-card. Member's cards are usually printed on paper with your logo and with a prepaid number of visits. In the configuration settings, you enter the minutes per visit for each solarium. The more flexible option is the chip-card. The chip-cards have small memory and they can be refilled many times with different amount of money. You can set few steps with discount prices for the chip-card users.  Using the chip-cards, your clients can choose any duration of the tanning session like with cash payment. There is a way to combine any payment methods. If the amount in the member's card or chip-card is not sufficient for the session, the rest can be paid in cash (but without the discount).

Daily reports are usually important for the administration of the studio. A daily report represents the tanning sessions, the cosmetics and the cards sold. It can be printed or saved to excel file or sent to an e-mail.

Statistics. Using the information from the daily reports, there are 18 different statistics which can be viewed for a selected period both in table view and in Bar Chart graph. 

Advertisements. If your studio has a second monitor connected to your computer (using dual screen configuration) you can place an animated Flash banner in the bottom of the screen with an advertisement or representation of some brands of your cosmetic or your animated logo. 

Second screen. With dual screen configuration, the second monitor (usually 32" or bigger) can be placed in the waiting saloon of your tanning studio and visitors can view the pictures of the solariums, the prices for a tanning session, each solarium state (busy, cooling or free) and the advertisement at the bottom of the screen. They will not see any additional screens that the operator will see on his/her screen while navigating with the program.

2. Installation

The installation consists of two separate tasks - hardware installation and software installation.

Hardware installation. For the program to communicate with the solariums, there are small devices, called controllers for each solarium, which are connected instead of the coin operated timers. Then a 8 wire FTP or STP cable connects them to the adapter box, mounted near the PC, which transfers the signals to the standard RS232 COM port of your PC. The common port is COM2, but it can be changed either trough windows's control panels or at the program side in the configuration settings in it's INI file.

The software installation is from your installation CD. Just insert it in your CD and the installation will start automatically. If autorun is disabled in your system, open the CD folder and double click on the program's icon. The following dialogs are for the destination folder and type of installation. If you are not an expert, better use the defaults. In addition, the installer will launch the installers for the smart-card reader driver and for flash and shockwave players. You can skip these additional installations if you have them already installed on your PC. Otherwise, let them go, or you might not be able to run the program. In the first run, you should enter the admistrator's menu and change the default password, configure the settings for each solarium, like prices, names, description, communication channel for the controller, pre-session time, session time, minimum and maximum times. You can increase or decrease the number of solariums too, or change their picture.

 


3. The Screens

 

 

Initialization screen

Your main screen is what represents the most important information of the current state of the solariums. From this screen you can go to other screens, pressing on the pictures of the solariums or the labels at the bottom.

Secondary screen - This is usually a big screen in the saloon. There the clients can view the current status of the solariums, and at the bottom there is a subscreen with a flash movie, which you can replace with one of your choice. It is usually used for advertisements.

Tanning session duration. This screen is the first of 2 steps you go to start a solarium, clicking with the left button of the mouse on the pictures of the solariums. Here you choose the time for the session and can see some additional information about the solarium. Also you will receive a message if the UV lamps are for replacement.

Payment. This is the second screen that you go to start a solarium after choosing the session time. You have option to choose from the 3 methods of payment. If you want to combine them, click on cash payment at last. If you insert a chip card when you see this screen, you will see information about the remainder in the card both as money and as time for the selected solarium. After clicking on the OK button, the solarium will start and you will go to the main screen.

Emergency switch. This window is shown when you right-click a solarium picture. Here you can start or stop the solarium, prolong the pre time or do a test to the solarium for 1 minute to see if all UV lamps are O.K.

Password. If you type rapidly '1234' while in the main screen, you will be asked to enter the password. When you enter it, you can go to the system configuration menu screen.

4. Routine tasks

Starting a tanning session

 

            To start a session for a selected Solarium, just click with the mouse on it's picture. You will be driven to the tanning session duration selection screen. Choose the desired time for the session and press OK button. The next screen is for selection of payment method. There are 3 ways of payment, which can be combined - Member's card, Chip-card and Cash payment. If your client has a chip-card, insert it while payment selection screen is visible and there will be visible the information form the card - the name of the client, remaining time and remaining money. To select payment, press the appropriate button, and to confirm the payment, press the OK button.

 

Selling cosmetics.

To sell cosmetics, go to the cash desk by pressing the CASH button in the main screen. Double click an item to add it to the shopping basket.

 

Selling/Refilling cards.

To review or refill a chip card, press CARDS button in the main screen, then select Chip cards. From the next screen you can work with chip cards. You can also print reports for a given customer or for all clients. The report includes information for all visits of the selected client.

 

View/print daily report.

To view the daily report press REPORT button in the main screen. If you have previously entered a password in the password form, you will be able to select another date for the report. The default date is the current date. You can also print the report, export it to excel table as file (this is done automatically, but the file is deleted next time the program is started), and also automatically send the excel file to an e-mail, if it is configured in the settings.

 



5. Administration

 

The database is a single file, located in the DATA folder in the program's directory (by default c:\Solarstudion1\data) and is named SOLARBAS.ABS. On closing, the program is making archive copies of the database with the current date included in the filename. On a database file crash, you can substitute it with the most recent archive.
